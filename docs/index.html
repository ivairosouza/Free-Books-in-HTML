Free Books in HTML

You are currently on a repository hosted on GitHub, a trusted and secure platform for developers. All files available here are safe, free of malicious scripts, and created using pure HTML and CSS. Feel free to browse and download without any concerns.

Hello! My name is Ivair Oliveira Souza. I'm currently learning English and discovered Gutenberg.org, a fantastic site offering free books in various formats, including EPUB, KINDLE, and HTML. Since I have some knowledge of HTML and CSS, I decided to download books in HTML format. However, I noticed that many files are not optimized for smartphones, with small fonts, excessive margins, poorly formatted indexes, and minimal styling.

To address this, I decided to create optimized versions of these books, making them more readable on all devices. You can still find the original versions at Gutenberg.org, and audio versions at Archive.org. My versions include both text and audio (when available). Note that some books lack audio because I couldn't find a recording online, but this is rare.
